## Packing Checklist (Example)

This is an example **task list** using Markdown checkboxes.

### Essentials

- [x] Passport / ID
- [x] Wallet (cash + cards)
- [x] Phone & charger
- [ ] Travel documents (boarding passes, reservations)
- [ ] Travel insurance information

### Clothing

- [ ] Shirts / tops
- [ ] Pants / shorts
- [ ] Underwear & socks
- [ ] Comfortable walking shoes
- [ ] Jacket or hoodie

### Toiletries

- [ ] Toothbrush & toothpaste
- [ ] Deodorant
- [ ] Comb/brush
- [ ] Travel-size shampoo / soap

## Simple “Before You Go” Checklist

- [ ] Check passport expiration date
- [ ] Check visa requirements (if international)
- [ ] Confirm baggage rules with airline
- [ ] Screenshot important reservations
- [ ] Share itinerary with a friend/family member
